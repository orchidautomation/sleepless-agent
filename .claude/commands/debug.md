# Debug Issue

Investigate and fix an issue in the Sleepless Agent codebase.

$ARGUMENTS

## Instructions

1. **Reproduce/Understand**
   - Identify where the issue likely occurs
   - Read relevant code paths
   - Check logs if available

2. **Diagnose**
   - Trace the execution flow
   - Identify root cause
   - Check for related issues

3. **Fix**
   - Implement minimal fix
   - Avoid unnecessary refactoring
   - Add error handling if appropriate

4. **Verify**
   - Test the fix
   - Check for side effects
   - Run related tests

5. **Document**
   - Add comments explaining non-obvious fixes
   - Note any gotchas discovered

Focus on fixing the actual issue, not rewriting surrounding code.
